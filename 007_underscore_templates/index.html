LearningBackbone - Underscore's Templates - App

Purpose of this example is to study how to handle Underscore's template strings.
We will try 4 methods to handle templates:

  1. hard coded in a View object
  2. hard coded in a <script> tag placed in the head of this document
  3. loaded via ajax from a external template file (using RequireJS text plugin
  4. fetched from a DOM element in this document
